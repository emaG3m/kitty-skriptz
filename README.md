# Kitty Skriptz

can i haz teh etherz?

### Setup

[Install Solidity](http://solidity.readthedocs.io/en/develop/installing-solidity.html)

Check `solc --version` for 0.4.18 or greater.

```
$ yarn install
```

### Build

Run solc build. Outputs abi and bin files to build/

```
$ yarn build
```

### Run

```
$ yarn start
```
